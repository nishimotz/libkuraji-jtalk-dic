# eng-dic-source

`cmudict-subset.dict` is a vendored, filtered subset (39,419 entries) of two
third-party, permissively-licensed sources, used to generate
`nvdajp-eng-dic.csv` (English word readings for JTalk speech / libkuraji
braille, replacing the GPL-licensed `bep-eng.dic` that nvdajp itself uses —
see the top-level README).

## Sources

- **[CMUdict](https://github.com/cmusphinx/cmudict)** (Carnegie Mellon
  University, permissive BSD-like license, see `LICENSE-cmudict.txt`):
  provides the ARPAbet pronunciation for each word.
- **[hermitdave/FrequencyWords](https://github.com/hermitdave/FrequencyWords)**
  `en_50k.txt` (MIT license, see `LICENSE-FrequencyWords.txt`), derived from
  OpenSubtitles word frequencies: used only to select which ~39k of
  CMUdict's ~135k entries to include (alphabetic words only, present in
  both lists), matching the rough vocabulary size of the GPL bep-eng.dic it
  replaces without pulling in CMUdict's long tail of proper nouns and rare
  technical/scientific terms.

## Format

One entry per line: `word PHONEME1 PHONEME2 ...` (ARPAbet, same format as
CMUdict itself). Generated by `eng_dic_maker.py`, which converts each
entry's phonemes to katakana via `arpabet_to_kana.py`.
